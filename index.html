Ejemplo de clases

Este es un ejemplo de cómo usar clases en HTML para aplicar estilos a elementos específicos. Las clases se definen en la sección CSS del documento y se aplican a los elementos utilizando el atributo "class".

En este ejemplo, se ha definido una clase llamada "titulo" que aumenta el tamaño de la fuente, la pone en negrita y la alinea al centro. La clase "parrafo" se utiliza para justificar el texto y añadir un margen superior.

Las clases son una forma útil de organizar los estilos CSS y aplicarlos a elementos específicos de una página web. Esto puede ayudar a mejorar la legibilidad y el mantenimiento del código.
